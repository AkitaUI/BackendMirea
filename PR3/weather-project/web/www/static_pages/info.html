О нашем погодном приложении

Это тестовое веб-приложение создано для демонстрации работы связки nginx + apache + php + MySQL.

nginx обслуживает статический контент (страницы вроде этой), а Apache + PHP — динамический (данные из базы, прогнозы и т.д.).

В качестве примера динамического контента вы можете перейти на: страницу списка городов.
